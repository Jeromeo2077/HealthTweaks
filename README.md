# HealthTweaks (Cyber Engine Tweaks mod)

This repository is laid out so you can copy/zip the `game/` folder directly into your Cyberpunk 2077 install.

## Folder layout

- `game/bin/x64/plugins/cyber_engine_tweaks/mods/HealthTweaks/` — CET mod root
  - `init.lua` — entry point loaded by CET
  - `modules/` — optional Lua modules
  - `config/` — config files (optional)
  - `data/` — data files (optional)

## Install (manual)

1. Ensure Cyber Engine Tweaks is installed.
2. Copy the contents of `game/` into your Cyberpunk 2077 install directory (merge folders).
